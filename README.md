# CGPA Calculator

A simple C++ project that calculates CGPA based on subject grades and credits.

## Features
- Accepts multiple subjects
- Takes grades and credits as input
- Calculates total grade points
- Displays final CGPA

## Technologies Used
- C++
- Standard Template Library (STL)

## How to Run

Compile:

```bash
g++ main.cpp -o cgpa
```

Run:

```bash
./cgpa
```

## Sample Input

Number of Subjects: 4

PHYSICS 4 A+
MATHEMATICS A 4
ENGLISH  3 B+
ELECTRONICS 3 A+

## Sample Output

CGPA = 8.29
